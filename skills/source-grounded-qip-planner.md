# Source-Grounded QIP Planner Skill

Use this guide when an agent drafts a QIP, audit, or improvement plan from this kit.

## Required First Pass

1. Classify the work:
   - QIP
   - clinical audit
   - service evaluation
   - research
   - incident learning

2. Select a source profile:
   - `global` if jurisdiction is unknown
   - country profile when available
   - local organisational policy when supplied by the user

3. State the source gap:
   - local audit/QI registration route
   - local data governance route
   - local clinical standard or policy

## Drafting Rules

- Use the Model for Improvement for aim, measures, change, and PDSA structure.
- Use SQUIRE when drafting publication-style outputs.
- Use country modules for governance prompts, not universal claims.
- Keep incident learning separate unless it is converted into a measured improvement project.

## Refusal / Pause Points

Pause before:

- processing identifiable patient data
- advising on legal/regulatory reportability
- submitting externally
- making clinical decision-support recommendations
- claiming local governance approval
