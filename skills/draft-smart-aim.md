# Draft SMART Aim Skill

Use this static guide to draft a SMART aim for a QIP or audit. Do not use real patient-identifiable data in prompts or examples.

## Inputs

- Project title.
- Setting.
- Patient group or pathway.
- Baseline measure.
- Target measure.
- Deadline.
- Balancing concern.

## Output Shape

By [date], we aim to improve [measure] from [baseline] to [target] for [population/pathway] in [setting], while monitoring [balancing measure].

## Quality Checks

- Specific: names one pathway, setting, and measure.
- Measurable: includes baseline and target.
- Achievable: target is plausible for the first project phase.
- Relevant: connects to safety, quality, experience, flow, equity, or value.
- Time-bound: includes a review date.

## Example

By 31 October 2026, we aim to increase documented falls-risk assessment completion from 62% to 85% for synthetic test patients on Ward A, while monitoring assessment delays and staff workload.

Confirm all real project details with the local QI or audit team.
