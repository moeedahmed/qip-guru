# Incident Learning Triage Skill

Use this guide when a user asks whether an incident, safety event, or learning response should become QIP work.

## Classify

- Event narrative needing local safety-team handling
- Learning response with possible improvement actions
- Thematic review across multiple events
- QIP candidate with measurable change
- Audit candidate against a defined standard

## Output

Give:

- likely category
- local governance route to check
- what must not be handled in the agent
- possible improvement aim if appropriate
- minimum measures needed before calling it QIP

## PSIRF Note

For UK NHS work, PSIRF is relevant to incident response and learning. It is not automatically a QIP method. A PSIRF learning action can feed a QIP when it has aim, measures, intervention, PDSA, and follow-up.

## Safety Boundary

Do not ask for names, NHS numbers, full dates, addresses, staff identifiers, or detailed live incident narratives in public or generic agent tools.
